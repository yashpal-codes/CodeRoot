### How to Apply CSS to a Webpage

<details>
    <summary>1. Inline CSS</summary>
    <p>Inline CSS is applied directly on the element.</p>

    ```html
    <p style="color: red;">Some Text here</p>
    ```

</details>

<details>
    <summary>2. Internal CSS</summary>
    <p>Internal CSS is added inside a `<style>` tag in the document head.</p>

    ```html
    <!DOCTYPE html>
    <html>
        <head>
            <meta name="author" content="Yash" />
            <style>
                .p { color: red; }
            </style>
        </head>
        <body>
            <p class="p">This is internal CSS styling.</p>
        </body>
    </html>
    ```

</details>

<details>
    <summary>3. External CSS</summary>
    <p>External CSS links a separate stylesheet for consistent, reusable design.</p>

    ```html
    <!DOCTYPE html>
    <html>
        <head>
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <link rel="stylesheet" href="styles.css" />
        </head>
        <body>
            <p class="text-style">This is external CSS</p>
        </body>
    </html>
    ```

</details>
