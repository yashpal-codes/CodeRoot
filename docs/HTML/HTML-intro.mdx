# HTML : Hyper Text Markup Language.

HTML is used to create web pages, It't like bonds in our body, it is skeleton of web page, while CSS is skin and other part of body.

## Global attributes

- **In HTML all tags supports Global attributes.**

1. **class** : Specify classname, using class attribute we can align classname for tags.
2. **id** : Specify id of an element.
3. **lang** : Specify the language code for the content in element.
4. **style** : Specify the _In-line_ style for an element.
5. **title** : Specify extra information about an element, used in tooltip.
6. **hidden** : Make an element hidden or visible
